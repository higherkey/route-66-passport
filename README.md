# Route 66 Digital Passport

Welcome to the **Route 66 Digital Passport**, a centennial project designed to help you explore "The Mother Road."

## Quickstart

### Prerequisites
- Node.js 18+
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

## Project Structure
- `/app`: Next.js App Router pages and layouts.
- `/components`: Reusable UI components (Stamps, Grid, etc.).
- `/hooks`: Custom React hooks (Geolocation, LocalStorage).
- `/public`: Static assets and PWA manifest.

For a detailed technical overview and roadmap, see [PROJECT.md](./PROJECT.md).
